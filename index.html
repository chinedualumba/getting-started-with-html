Some key concepts

Firstly: How the web works.

Learned about the communication between computers, servers and web browsers. How opening a web page sends a HTTP Request to a server which in turn finds the appropriate HTML document for that page, sends it back to the computer and is tranlated into the page we see by the web browsers we have e.g Chrome, Firefox and if you're a really patient person, Internet Explorer

The HTML Document is made of several things: Text content that we see on the page, HTML Markup which edits how the page looks, References to images and videos, and even links to other pages.

Secondly: How HTML Documents are created

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags.

Making text look different is one part of what HTML Markup is all about. HTML Markup consists of tags that appear in brackets like these < > - opening bracket and having a "/" - in the closing bracket ends that element. Several different tags make the text in the document look differently.

Examples are putting a "b" or an "em" in the tiny brackets. The first one makes everything in between the opening and closing bracket bold while the second one writes everything in italics.

Some tags are inline tags while some are block tags. Inline tag ends a line and starts the next words on a different line. While Block tags create an invisible block around the text. Pretty cool stuff and the further we go, the more I hope to learn more information that I can share.
